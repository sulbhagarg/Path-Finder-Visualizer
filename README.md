# Path-Finder-Visualizer

## Build and Run

### Prerquisites -
1. Install Node.Js and npm
   - Refer [here](https://nodejs.org/en/download/)

### Run Software

1. Clone Repo
   - `$> git clone https://github.com/sulbhagarg/Path-Finder-Visualizer.git`
2. Change Directory
   - `$> cd Path-Finder-Visualizer`
3. Install Dependencies
   - `$> npm install`
4. Start server
   - `$> npm start`
